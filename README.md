# ShapeExtrusion Pro

**ShapeExtrusion** is a premium, professional-grade 3D design tool built with **React** and **Babylon.js**. It enables users to transform 2D conceptual sketches into fully editable 3D volumes with real-time property manipulation and a modern, glassmorphic user interface.

## ✨ Core Features

### 🖌️ Intelligent Sketching
*   **Precision Drafting**: High-contrast workspace with precision snapping and point placement.
*   **Live Preview**: Real-time "rubber-band" line preview that follows your cursor for better accuracy.
*   **Auto-Extrusion**: Intelligently detect valid polygons and extrude with a single click.

### 🏗️ Advanced 3D Manipulation
*   **Dynamic Properties Inspector**: Real-time control over height scaling and color customization.
*   **Vertex Edge Manipulation**: Select and deform individual nodes in 3D space with soft-selection logic.
*   **Interactive Transform**: Move objects within the 3D scene using a unified move mode.

### 🎨 Premium Aesthetics
*   **Glassmorphic Design System**: A state-of-the-art UI utilizing backdrop blurs, subtle borders, and a deep dark-mode palette.
*   **Enhanced Rendering**: Point-light and hemispheric lighting setup for superior depth perception.
*   **Color Swatch Library**: Premium preset color palette alongside full HEX code support.

## 🚀 Quick Start

### Installation
1. Clone the repository:
   ```bash
   git clone [repository-url]
   ```
2. Install dependencies:
   ```bash
   npm install
   ```
3. Launch development environment:
   ```bash
   npm run dev
   ```

## ⌨️ Professional Shortcuts

| Key | Action |
|-----|--------|
| **D** | Toggle Sketch Mode |
| **M** | Toggle Move/Transform Mode |
| **V** | Toggle Vertex Editing Mode |
| **R** | Reset Viewport |
| **G** | Toggle Grid Visibility |
| **+ / -** | Zoom In / Zoom Out |

## 🛠️ Technical Stack
*   **Frontend**: React.js
*   **3D Engine**: Babylon.js
*   **Styling**: Vanilla CSS (Modern Design System)
*   **Math**: Earcut (Polygon Triangulation)
